
# FZF KEYBINDS (FISH)

## FILE / DIRECTORY

* Ctrl + F  : Cari file, buka di nvim
* Ctrl + P  : Cari folder, lalu cd
* Ctrl + Alt + F : Search directory

## HISTORY

* Ctrl + R  : History (fzf)
* Ctrl + H  : History (custom)

## PROCESS

* Ctrl + Alt + P : List process
* Ctrl + K       : Kill process (force)

## GIT

* Ctrl + Alt + L : Git log
* Ctrl + Alt + S : Git status

## SYSTEM

* Ctrl + V : Variables

## CATATAN

* Alt bisa diganti Esc + key jika tidak terbaca
* Semua keybind aktif di semua mode

## STATUS

* fzf aktif
* keybind aktif
* siap digunakan
